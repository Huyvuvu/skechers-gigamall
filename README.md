# 👟 SKECHERS Vietnam – Giga Mall Outlet

> E-commerce website for Skechers Vietnam at Giga Mall Outlet

🔗 **Live:** https://skechers-gigamall-outlet.vercel.app

## ✨ Features
- 🛍️ Full e-commerce UI (Women / Men / Kids / Technologies / Sale)
- 🔐 Role-based auth: User · Admin · Super Admin
- 📊 Admin Dashboard (Products, Orders, Media Upload, Analytics)
- 📱 Mobile-first, responsive design
- 🛒 Cart, Quick View, Wishlist, Checkout → Payoo

## 🚀 Deploy
```bash
git clone https://github.com/YOUR_USERNAME/skechers-gigamall.git
cd skechers-gigamall
vercel --prod
```
